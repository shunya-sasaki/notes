and references for various programming languages, tools, and concepts.

## 📦 Requirements

- mdBook

## ⚙️ Setup

To install mdBook, you can use the following command:

```bash
cargo install mdbook
```

## 🚀 Usage

To build english version of the book, run the following command:

```bash
mdbook build en
```

And to build japanese version of the book, run the following command:

```bash
mdbook build ja
```
