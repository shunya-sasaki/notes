## 💡 What is Recharts?

- Recharts は React 向けのコンポーザブルなチャートライブラリで、SVG でチャートをレンダリングし、内部で D3 にインスパイアされたユーティリティを活用しています。
- 宣言的なコンポーネントベースの API を提供しており、LineChart、BarChart、AreaChart、PieChart、ScatterChart、RadarChart、RadialBarChart、ComposedChart、XAxis、YAxis、CartesianGrid、Tooltip、Legend、Brush、ReferenceLine などのパーツを組み合わせてチャートを構築できます。
- dataKey プロパティを通じてシンプルなデータ（オブジェクトの配列）を受け取り、React の state、props、hooks と自然に統合できます。
- レスポンシブコンテナ、アニメーション、組み込みのインタラクティブ機能（ツールチップ、凡例、ブラッシング、ハイライト）を含みます。
- 高いカスタマイズ性: レンダラーのオーバーライド、カスタムシェイプ/ラベルの提供、レンダーコールバックを使用したビジュアルの調整が可能です。
- TypeScript の型定義と充実したドキュメントが付属しており、MIT ライセンスのオープンソースです。
- 小～中規模のデータセットを使用したダッシュボードやレポートに最適です。数百万ポイントの場合は、canvas/WebGL ベースの代替手段を検討してください。
